# AI Code Quality Analyzer

An AI-inspired web application built using Python and Flask that analyzes Python code quality and generates automated improvement suggestions.

---

## Features

- Code quality scoring system
- Debug print statement detection
- Long line detection
- Function count analysis
- Warning generation system
- Interactive dark-themed dashboard
- Automated maintainability insights

---

## Tech Stack

- Python
- Flask
- HTML
- CSS

---

## Project Workflow

1. User pastes Python code
2. Flask backend processes code
3. Rule-based analyzer checks code quality
4. Warnings and suggestions generated
5. Dashboard displays quality score and metrics
